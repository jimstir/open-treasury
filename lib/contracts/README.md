# Open Treasury Smart Contracts

The smart contracts for the Open Treasury project.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Compile contracts:
   ```bash
   npx hardhat compile
   ```

3. Run tests:
   ```bash
   npx hardhat test
   ```


## Dependencies

- OpenZeppelin Contracts
- Hardhat (development)
- Hardhat Toolbox (testing and deployment utilities)
